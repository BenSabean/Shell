OS_Project
==========

A shell based Project for Operating Systems


Features
==========

1. PIPING
2. HISTORY OF COMMANDS
3. TAB COMPLETION
4. REDIRECTION
5. PLACE COMMANDS IN THE BACKGROUND
6. MOVE COMMANDS INTO THE FOREGROUND
7. SUPPORT BUILT IN COMMANDS
8. KEYBOARD SHORTCUTS
